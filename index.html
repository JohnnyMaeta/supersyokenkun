通知表所見アシストの使い方

このツールは、先生方が通知表の所見を書く作業をお手伝いするためのものです。いくつかのキーワードを入れるだけで、AIが自然な文章を作成してくれます。ICTが苦手な先生でも、簡単なステップで使えるように設計されていますので、ご安心ください。

1. 最初の準備（初回のみ）

初めて使うときだけ、以下の2つの準備が必要です。

ステップ1： APIキーを取得して設定する

AIを使うための「パスワード」のようなものである「APIキー」を取得します。

  1. Google AI Studio にアクセスし、Googleアカウントでログインします。
  2. 「Create API key in new project」というボタンをクリックして、新しいキーを作成します。
  3. 作成されたキー（長い文字列）をコピーします。
  4. スプレッドシートに戻り、メニューの「所見支援」→「サイドバーを開く」をクリックします。
  5. サイドバー下部の「設定」にある入力欄に、コピーしたAPIキーを貼り付け、「APIキーを保存」ボタンを押します。

これでAIと通信できるようになりました。

ステップ2： 「私の文体」をAIに学習させる

AIが先生の書き方に似た、自然な文章を作れるように、先生の文体を学習させます。

  1. スプレッドシートのメニューから「所見支援」→「文体サンプル用シートを作成/表示」を選びます。
  2. 文体サンプル という名前のシートが作成されます。
  3. A列に、過去に先生自身が作成した所見を10個〜20個ほど、1つのセルに1件ずつ貼り付けます。（個人名などのプライバシー情報は削除してください）
  4. サイドバーにある「私の文体を分析する」ボタンをクリックします。
  5. 分析が終わると、文体プロファイル というシートが自動で作成されます。これでAIの学習は完了です。

2. 所見の作り方

準備が終わったら、いつでも所見を作ることができます。

ステップ1： 箇条書きメモを入力する

サイドバーの「箇条書きメモ」の欄に、所見に書きたい生徒の様子を、短い言葉で入力します。Enterキーで改行することで、複数項目を入力できます。

例：

  • 学習への姿勢が安定してきた
  • グループ活動で友達の話をよく聞ける
  • 苦手な計算問題も、諦めずに取り組むようになった
便利なヒント： スプレッドシートのセルにメモが入力されている場合、そのセル（複数選択も可能）を選んでから、サイドバーの「選択セルの内容を取得」ボタンを押すと、メモ欄に内容を簡単に取り込めます。

ステップ2： ゴールや文字数を設定する

  • ゴール： 作成する文章の目的（生徒の成長を伝えたい、保護者を安心させたいなど）を選びます。
  • およその文字数： 作成したい文章の長さを指定します。
  • 対象学年： 文章の言葉遣いを調整するために、対象の学年を選びます。

ステップ3： 所見を生成して書き出す

  1. 所見を書き出したいセルを、スプレッドシート上でクリックして選択します。
  2. サイドバーの「所見を生成」ボタンを押します。
  3. 少し待つと、AIが作成した文章が、選択していたセルに自動で入力されます。

あとは、必要に応じて手直しをすれば完成です！
